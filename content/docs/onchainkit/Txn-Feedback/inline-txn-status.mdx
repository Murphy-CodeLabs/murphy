## Status States

### Success State

- **Icon**: CheckCircle (green)
- **Text**: "Success"
- **Use**: Completed transactions
- **Color**: Green theme

### Error State

- **Icon**: XCircle (red)
- **Text**: "Failed"
- **Use**: Failed transactions
- **Color**: Red theme

### Processing States

- **Icon**: Loader2 (blue, animated)
- **Text**: "Processing"
- **Use**: preparing, signing, sending, confirming
- **Color**: Blue theme
- **Animation**: Spinning loader

### Idle State

- **Icon**: Clock (gray)
- **Text**: "Pending"
- **Use**: Queued or idle transactions
- **Color**: Gray theme

## Size Variations

### Small (sm)

- Icon: 12px (w-3 h-3)
- Text: text-xs
- Use: Table cells, compact lists

### Medium (md) - Default

- Icon: 16px (w-4 h-4)
- Text: text-sm
- Use: General purpose, cards

### Large (lg)

- Icon: 20px (w-5 h-5)
- Text: text-base
- Use: Prominent displays, headers
