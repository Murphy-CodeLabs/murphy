## Installation

<Steps>
  <Step>
    Install dependencies

    Start by installing required Metaplex Core dependencies

    <DependenciesInstallCommands packageUrl={"@solana/web3.js @solana/wallet-adapter-react @metaplex-foundation/umi @metaplex-foundation/umi-bundle-defaults @metaplex-foundation/mpl-core @metaplex-foundation/umi-signer-wallet-adapters react-hook-form @hookform/resolvers zod"} />
  </Step>

  <Step>
    Add Wallet Provider

    Make sure you have added the Wallet Provider to your application. If not, follow the steps in the [Connect Wallet Button](/docs/onchainkit/connect-wallet-button) guide first.

  </Step>

  <Step>
    Add Core Asset Launchpad

    <InstallationCommands
      packageUrl={`${process.env.NEXT_PUBLIC_BASE_URL}/r/core-asset-launchpad.json`}
    />

  </Step>
</Steps>
