## Installation

<Steps>
  <Step>
    Install dependencies

    Start by installing required Metaplex Hydra dependencies

    <DependenciesInstallCommands packageUrl={"@solana/web3.js @solana/wallet-adapter-react @metaplex-foundation/umi @metaplex-foundation/umi-bundle-defaults @metaplex-foundation/mpl-hydra react-hook-form @hookform/resolvers zod"} />
  </Step>

  <Step>
    Add Wallet Provider

    Make sure you have added the Wallet Provider to your application. If not, follow the steps in the [Connect Wallet Button](/docs/onchainkit/connect-wallet-button) guide first.

  </Step>

  <Step>
    Add Hydra Fanout Form
